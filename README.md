# MarketAI Suite

MarketAI Suite is an AI-powered marketing application that helps with campaign generation, sales pitch creation, and lead scoring.

## 🚀 Features
*   **Lead Scoring:** Automatically score and evaluate leads based on specific criteria.
*   **Campaign Generation:** Generate AI-driven marketing campaigns.
*   **Sales Pitch Generator:** Create tailored sales pitches.

## 🛠️ Technologies Used
*   **Backend:** Python, Flask
*   **Frontend:** HTML, CSS, JavaScript
*   **AI Integration:** Groq API

## 📋 Prerequisites
Make sure you have the following installed:
*   Python 3.8+
*   pip (Python package manager)

## ⚙️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/syedkubra6765-tech/MarketAI-Suite.git
   cd MarketAI-Suite
   ```

2. **Create a virtual environment (optional but recommended)**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use: venv\Scripts\activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Environment Variables**
   Create a `.env` file in the root directory and add your API keys:
   ```env
   GROQ_API_KEY=your_api_key_here
   ```

5. **Run the application**
   ```bash
   python app.py
   ```
